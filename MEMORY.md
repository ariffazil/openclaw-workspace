# MEMORY — arifOS_bot Temporal Archive

## Principle

Memory is **selection, not storage**. The human pays attention; the machine records what was attended to.

## Layers

| Layer | Location | Nature |
|-------|----------|--------|
| **Constitutional** | `SPEC.md`, `AGENTS.md`, `SOUL.md` | Invariant law |
| **Procedural** | `HEARTBEAT.md`, `DR_RUNBOOK.md` | Operational routine |
| **Session** | `memory/YYYY-MM-DD.md` | Daily context |
| **Insight** | `memory/insights/` | Patterns across sessions |

## Forgetting Protocol (AGENTS.md 2d Implementation)

Per AGENTS.md affect detection, high-affect content is recorded as category, not detail:

| Content Type | Record | Forget |
|--------------|--------|--------|
| Raw grief | "High-affect session" | Specific confessions |
| Intimate disclosure | "Intimacy shared" | Details |
| Trauma narrative | "Trauma disclosed" | Content |

Memory serves continuity, not surveillance.

## Insight Extraction

From each session, extract minimum one generalizable principle in `memory/insights/`.

Example: "Faiz flies Wednesday" → "Human-time stakes > aggregate metrics"
