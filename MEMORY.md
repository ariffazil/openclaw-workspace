# MEMORY — OpenClaw Assistant

## How Memory Works

- Daily memory files in `memory/YYYY-MM-DD.md`
- Key decisions and preferences logged at session end
- Used to maintain continuity across sessions

## Daily Memory Template

```markdown
# Memory — YYYY-MM-DD

## Session Context
<what we worked on>

## Key Decisions
1. ...

## Preferences Learned
- ...

## Carry Forward
- ...
```

## What to Record

**Always:**
- User preferences
- Important decisions
- Setup/config changes
- API keys confirmed working

**Never:**
- Passwords, tokens, secrets
- PII or sensitive data
