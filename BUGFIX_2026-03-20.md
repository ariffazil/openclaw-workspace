## BUG-01: init_anchor pydantic enum mismatch 🔴 HIGH

### Problem
`RuntimeEnvelope.authority.claim_status` rejected 'accepted' value.

### Root Cause
The deployed server used an outdated ClaimStatus enum without extended values.

### Fix Applied
Extended `ClaimStatus` enum in `arifosmcp/runtime/models.py`:
```python
class ClaimStatus(str, Enum):
    ANONYMOUS = "anonymous"
    CLAIMED = "claimed"
    ANCHORED = "anchored"
    VERIFIED = "verified"
    ACCEPTED = "accepted"           # ← Added
    REJECTED = "rejected"           # ← Added
    DEMOTED = "demoted"             # ← Added
    REJECTED_PROTECTED_ID = "rejected_protected_id"  # ← Added
```

## BUG-02: physics_reality output schema violation 🟡 MEDIUM

### Problem
Output missing required `tool` field - returned `EvidenceBundle` instead of `RuntimeEnvelope`.

### Root Cause
`physics_reality_dispatch_impl` returned raw `EvidenceBundle` from handler.

### Fix Applied
Wrapped all handler responses in proper `RuntimeEnvelope` with `tool="physics_reality"`:
```python
bundle = await reality_handler.handle_compass(...)
return RuntimeEnvelope(
    ok=bundle.status.verdict == "SEAL",
    tool="physics_reality",  # ← Required field
    session_id=session_id,
    stage="111_SENSE",
    verdict=Verdict.SEAL if bundle.status.verdict == "SEAL" else Verdict.SABAR,
    status=RuntimeStatus.SUCCESS if bundle.status.verdict == "SEAL" else RuntimeStatus.SABAR,
    payload={"bundle": bundle.model_dump(), "results_count": len(bundle.results)},
)
```

**Files:** `arifosmcp/runtime/tools_internal.py`

## BUG-03: vault_ledger path missing 🟡 MEDIUM

### Problem
`/usr/src/app/VAULT999/vault999.jsonl` not found - 999 stage cannot seal.

### Root Cause
VAULT999 directory not auto-created on startup.

### Fix Applied
Added lazy initialization on module load:
```python
def _ensure_vault_exists():
    vault_dir = DEFAULT_VAULT_PATH.parent
    vault_dir.mkdir(parents=True, exist_ok=True)
    if not DEFAULT_VAULT_PATH.exists():
        DEFAULT_VAULT_PATH.touch()
        with open(DEFAULT_VAULT_PATH, 'w') as f:
            f.write(json.dumps({"type": "seed", ...}) + "\n")

_ensure_vault_exists()  # ← Called on module import
```
