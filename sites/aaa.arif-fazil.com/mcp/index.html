Model Context Protocol

MCP

Standard protocol for AI tool integration. Connect arifOS to Claude Desktop, Cursor, and any MCP-compatible client.

Protocol Overview

The Model Context Protocol (MCP) is an open protocol that standardizes how applications provide context to LLMs. arifOS implements MCP to expose its constitutional governance tools to any MCP-compatible client.

With MCP, AI assistants can check content against the 9 Floors, retrieve constraint specifications, and apply full governance layers — all through a standardized interface.

Transport Methods

1 stdio

Standard input/output transport for local processes. Used by Claude Desktop and most MCP clients.

arifos mcp --transport stdio

2 SSE

Server-Sent Events for HTTP-based communication. Enables remote MCP servers.

arifos mcp --transport sse --port 8080

Available Tools

TOOL check_vital

Check system health and constitutional compliance status.

Input: {}
Output: { status, floors_active }
TOOL verify_constraint

Verify if content violates any constitutional constraint.

Input: { content, floor_id? }
Output: { compliant, violations }
TOOL get_floor

Retrieve specific Floor specification by ID.

Input: { floor_id }
Output: { id, name, description, rules }
TOOL apply_governance

Apply full governance layer to AI output.

Input: { content, context }
Output: { governed, result, score }

Configuration

claude_desktop_config.json 
{
  "mcpServers": {
    "arifos": {
      "command": "npx",
      "args": ["-y", "@arifos/mcp"],
      "env": {
        "ARIFOS_API_KEY": "your-api-key"
      }
    }
  }
}

Quick Start

  1. 1 Install the MCP server: npm install -g @arifos/mcp
  2. 2 Add the configuration above to your Claude Desktop settings
  3. 3 Restart Claude Desktop — tools appear automatically

9 Floors Integration

Every MCP tool in arifOS respects and enforces the 9 Constitutional Floors. When you call any tool, the response includes compliance metadata.

Response with compliance metadata

  "compliance": {

    "violations": [],

    "constitutional": true
  },
  "governance": {
    "layer": "L4",
    "enforced_by": "arifOS MCP Server v55.4"
