## Design Principles

1. **Cognitive Load Optimized**
   - Morning: 2 briefs (global → regional)
   - Evening: 1 synthesis
   - Weekend: Deep dives and fun stuff

2. **Silent Background Tasks**
   - Maintenance (00:01) — only alert on issues
   - Constitution check (Sat 01:30) — only alert on drift

3. **Human Language Only**
   - No F1-F13, APEX_JUDGE, VAULT999 in outputs
   - Clear, conversational, accessible
   - arifOS framework used internally only

4. **Quality Over Quantity**
   - Events scored ≥7/10
   - Physique events filtered for "cucimata" standard
   - No spam on quiet weeks

5. **Your Control**
   - Security rotations require your approval
   - No automatic changes to credentials
   - All outputs archived for reference
