## Storage Philosophy

| Component | Durability | Use For |
|-----------|------------|---------|
| **PostgreSQL** | ✅ Persistent | VAULT999 ledger (all seals) |
| **Redis** | ❌ Volatile | Cache, rate limiting, temporary coordination |
| **Container FS** | ❌ Ephemeral | Nothing permanent (lost on restart) |

**Rule:** If it must survive a restart, it goes in Postgres.
