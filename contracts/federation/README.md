## 111 Federation Contract

This directory holds the canonical **111 SENSE** contract for AAA federation.

`111-sense.yaml` defines what is honestly live in this repo:

- public A2A discovery surfaces
- trust tiers and auth lanes
- governed ingress posture
- the boundary between static publication and runtime bridge execution

The contract exists to keep AAA from claiming live A2A message handling when this repository is only publishing discovery assets.
